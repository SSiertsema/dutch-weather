# Dutch weather

Inform the user on current weather, how to dress and inspire what they could do. If interested the user can also view a forecast for the upcoming 5 days.

## Getting Started

To install dependencies:

```
npm install
```

To run the application:

```
npm start
```

### Prerequisites

node.js > v6.*
